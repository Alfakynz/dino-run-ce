# Dinosaur Run CE

A clone of Google Chrome's dinosaur game (from the no internet screen) for the TI-83/84 Plus CE graphing calculator.
Original [game](https://github.com/commandblockguy/dino-run-ce) by [commandblockguy](https://github.com/commandblockguy)

## Installation

#### Install TI-Connect CE or TiLP

For **Linux**: `sudo apt install tilp2`
For **MacOS/Windows**: install the app from the [website](https://education.ti.com/fr/produits/logiciel-ordinateur/ti-connect-ce-sw)

#### Install the game

- Download `DINO.8xp` from the [release](https://github.com/Alfakynz//releases/latest)
- Send `DINO.8xp` to the calculator using TI-Connect CE or TiLP

#### Install the C libraries

- Download the [C libraries](https://github.com/CE-Programming/libraries/releases)
- Send `clibs.8xg` to the calculator using TI-Connect CE or TiLP

#### OS 5.5

You will need to use [arTIfiCE](https://yvantt.github.io/arTIfiCE/) to run native programs.

## Play

Select DINO from the prgm menu and press enter or select Dino from the Cesium app.

## Controls
UP/2nd to jump, DOWN/Alpha to duck / fall, enter/2nd to restart, clear to exit.
